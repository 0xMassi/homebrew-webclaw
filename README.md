# homebrew-webclaw

Homebrew tap for [webclaw](https://github.com/0xMassi/webclaw) — the fastest web scraper for AI agents.

## Install

```bash
brew tap 0xMassi/webclaw
brew install webclaw
```

This installs two binaries:
- `webclaw` — CLI for extracting web content
- `webclaw-mcp` — MCP server for AI agents (Claude, Cursor, Windsurf)

## Upgrade

```bash
brew upgrade webclaw
```
